#

# - re (expresiones regulares)
# - json (manejo de JSON)
# - pathlib (manejo de rutas)
# - datetime (fechas y tiempos)
# - argparse (argumentos CLI)
# - typing (type hints)
# - subprocess (ejecución de comandos) - NUEVO v2.0
# - shutil (utilidades de shell) - NUEVO v2.0
#
# ============================================================================
# HERRAMIENTAS EXTERNAS (v2.0 - para funcionalidad de escaneo)
# ============================================================================
#
# Estas herramientas son necesarias SOLO si usas --scan
# NO son necesarias para análisis de archivos existentes
#
# Instalación en Debian/Ubuntu/Kali:
# -----------------------------------
# sudo apt update
# sudo apt install -y nmap nikto gobuster curl
#
# Instalación en Fedora/RHEL/CentOS:
# -----------------------------------
# sudo dnf install -y nmap nikto gobuster curl
#
# Instalación en Arch Linux:
# ---------------------------
# sudo pacman -S nmap nikto gobuster curl
#
# Verificar instalación:
# ----------------------
# which nmap nikto gobuster curl
# nmap --version
# nikto -Version
# gobuster version
# curl --version
